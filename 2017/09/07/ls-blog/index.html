一、举例

1. git init  
2. touch login.txt (进行编辑)；
3. git add .-->git commit -m 完成登录页面
4. git branch car
5. git  checkout car
6. touch car.txt (进行编辑)--> git add .-->git commit -m 购物车编辑
7. git  checkout master(car.txt 将会看不见，因为处于car分支下面)；
8. --> git add .-->git commit -m 解决master问题
9.  git  checkout car
10.   car.txt (进行编辑)--> git add .-->git commit -m 购物车完成
11.    git  checkout master
12.    git merge car -m 备注信息  （将指定分支合并到当前分支下）
13.    git branch -d car 删除指定的分支
